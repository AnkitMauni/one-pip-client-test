The data feed delivers quotes from Universal DDE Connector. This specialized utility enables the collection of quotes from different data sources that support the DDE (Dynamic Data Exchange) protocol.

This native solution is built into the platform, which ensures minimal delays on quote delivery. To speed up data transfer, the data feed operates in multiple threads.

Pre-setup

Download and install Universal DDE Connector from MetaQuotes Support Center.

Data feed setup
  • Feed server. Specify the adddress of the server where the Universal DDE Connector is installed, and a port for connection, separated by a colon.
  • Feed login. Specify here the account that was previously created in Universal DDE Connector
  • Password. Universal DDE Connector account password
  • Synthetic quotes. MetaTrader 5 UniFeeder enables the delivery of quotes for the symbols on which Universal DDE Connector does not broadcast any data. The relevant data is generated by the mathematical conversion of other symbols' quotes available in the Connector.

Specify the quote calculation formula on the Parameters tab:

  • Parameter is the name of the target symbol and its price. For example, GOLDVND.ask or GOLDVND.bid
  • Value is the formula for calculating quotes. For example, GOLD.ask 195050 or GOLD.bid 195050. The formula can include any symbol, quotes for which are available at Universal DDE Connector, but only one such symbol can be used for one formula. The formula supports multiplication ("), division (/), addition (+) and subtraction (-), as well as parentheses () to determine the order of calculation
Platform setup
  • Enable the data feed and check the journal for the connection status.
